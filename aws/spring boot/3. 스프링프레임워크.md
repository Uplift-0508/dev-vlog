## 빈 + 컨테이너
IoC 를 통해 어떻게 의존도를 낮췄을까

스프링 MVC 는
Front Controller 패턴에 Spring 의 의존성 주입을 이용해서 컴포넌트들의 생명주기를 관리할 수 있는 컨트롤러 중심의 웹 MVC 프레임워크다.
