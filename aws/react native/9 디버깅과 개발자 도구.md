# 디버깅과 개발자 도구

## 자바스크립트 디버깅 방법

### 개발자 옵션 활성화
앱 내부 개발자 메뉴를 통해 크롬 개발자 도구를 활성화합니다. 
iOS 시뮬레이터에서는 커멘드+D 를 누르면 됩니다. 안드로이드 에뮬레이터는 커맨드+M (Mac일 경우) 혹은 컨트롤+M(윈도우일 경우) 을 누르면 됩니다. 
Expo 앱의 경우도 단축키가 같습니다.
